What is Cisco PSIRT?

The Cisco Product Security Incident Response Team (Cisco PSIRT) is a dedicated, global team that manages the receipt, investigation, and public reporting of security vulnerability information that is related to Cisco products and networks. PSIRT investigates vulnerabilities across the entire Cisco product portfolio assigning Common Vulnerabilities and Exposures (CVE) identifiers to internally and externally found vulnerabilities across hundreds of products reflecting Cisco's commitment to transparency and helping customers mitigate risk.

PSIRT investigates all reports, regardless of the Cisco software code version, through the last day of support of a given product. Issues will be prioritized based on the potential severity of the vulnerability and other environmental factors. Ultimately, the resolution of a reported incident may require upgrades to products that are under active support from Cisco.

Thus, PSI is your friendly Google search engine but for Cisco advisories.
Cisco PSIRT supports the following industry-wide security standards:
  • CVE
  • CVSS
  • CWE
  • CSAF
  • CVRF
A standard maintained by MITRE that identifies, defines, and catalogs publicly disclosed cybersecurity vulnerabilities. More information at: https://cve.org (Old website https://cve.mitre.org)
Common Vulnerability Scoring System (CVSS) provides a way to capture the principal characteristics of a vulnerability and produce a numerical score reflecting its severity. More information at: https://first.org/cvss
Common Weakness Enumeration (CWE) is a community-developed list of common software security weaknesses. More information at: https://cwe.mitre.org
Common Security Advisory Framework (CSAF) is a specification for structured machine-readable vulnerability-related advisories and further refine those standards over time. CSAF is the new name and replacement for the Common Vulnerability Reporting Framework (CVRF). More information at: https://csaf.io
Common Vulnerability Reporting Format (CVRF) is an industry standard format for publishing security advisories in machine readable format. CSAF is the industry standard that replaces CVRF. Cisco will support CVRF until December 31, 2023. More information at: https://www.first.org/newsroom/releases/20210601

About PSIRT

The application takes advantage of the Cisco PSIRT RESTful API to search for CVEs and show all its data and related information from other sites not only to easily identify the CVE and its workaround (if provided), but also to get all possible solutions at one and centralized place and to stay up to date with any security vulnerability.

PSIRT is developed using the Cisco UI/UX Kit that is used to design and develop applications with Cisco User Interface standards to optimize the user experience. Cisco UI Kit is a modern, lightweight, responsive and programmable presentation layer that can be used to style HTML primitive elements.

Currently, supported network operating systems are:

  • Cisco IOS Software.
  • Cisco IOS XE Software.
  • Cisco NX-OS Software.
  • Cisco NX-OS in ACI mode Software.
  • Cisco Adaptive Security Appliance (ASA) Software.
  • Cisco Firepower Threat Defense (FTD) Software.
  • Cisco Secure Firewall Management Center (FMC) Software.
  • Cisco Firepower eXtensible Operating System (FXOS) Software.
